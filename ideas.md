# 股票操作戰略儀表板 - 設計理念

## 選定設計方案：現代金融專業主義風格

### 設計理念
**Modern Financial Professionalism** - 融合現代金融科技的專業感與易用性，打造一個既嚴肅又親近的儀表板體驗。

### 核心原則
1. **數據優先**：視覺層次清晰，重要指標突出，輔助信息適度退後
2. **可信度設計**：深色背景搭配精確的數據呈現，傳達專業與穩定感
3. **實時感受**：動態圖表與漸變色彩暗示市場的流動性與變化
4. **易於掃讀**：卡片式佈局、清晰的顏色編碼（綠漲紅跌），讓投資者快速判斷

### 色彩哲學
- **主色調**：深藍 (#1e3a8a) 與深灰 (#1f2937)，傳達專業與信任
- **強調色**：翠綠 (#10b981) 表示上升/正向，紅色 (#ef4444) 表示下降/警示
- **背景**：深色模式 (dark background)，減少眼睛疲勞，突出數據
- **文字**：淺灰白色，確保高對比度與可讀性

### 佈局範式
- **網格系統**：2x2 或 3x2 卡片佈局，每個卡片代表一個關鍵指標
- **非對稱元素**：頂部大型指標展示，下方補充數據，避免單調的網格感
- **側邊欄**：可選的數據詳情面板，提供深度分析而不打亂主視圖

### 簽名元素
1. **漸變卡片邊框**：微妙的漸變邊框暗示市場動向
2. **迷你圖表**：每個指標卡片內嵌小型走勢圖，視覺化趨勢
3. **狀態指示器**：彩色圓點或小圖標表示指標狀態（安全/警示/危險）

### 互動哲學
- **懸停效果**：卡片輕微上浮、邊框發光，暗示可互動性
- **平滑過渡**：數據更新時的淡入淡出動畫，避免突兀變化
- **實時更新提示**：微妙的脈衝動畫或時間戳記，傳達數據的即時性

### 動畫指南
- **進入動畫**：卡片從下方淡入並上升（300ms easing: ease-out）
- **數據更新**：數值變化時的輕微縮放 + 顏色閃爍（200ms）
- **懸停狀態**：卡片邊框發光、背景微亮（150ms transition）
- **警示動畫**：融資維持率低於安全線時的溫和脈衝（1s infinite）

### 排版系統
- **標題字體**：Poppins (bold, 700) - 現代、幾何感，適合金融科技
- **正文字體**：Inter (regular, 400) - 清晰易讀，專業感
- **數據字體**：Roboto Mono (600) - 等寬字體，強調精確性
- **字級層級**：
  - 大標題：28px (Poppins 700)
  - 卡片標題：16px (Poppins 600)
  - 數值：24px (Roboto Mono 600)
  - 輔助文字：12px (Inter 400)

### 視覺特徵
- **陰影**：多層陰影 (0 10px 30px rgba(0,0,0,0.3))，增加深度感
- **圓角**：8px 標準圓角，12px 卡片邊角，傳達現代感
- **邊框**：1px 半透明白色邊框 (rgba(255,255,255,0.1))，增加層次感
- **背景紋理**：微妙的噪點或網格紋理，避免純平面感
